# Cocos Creator Implementation

This folder contains a minimal rewrite of the Greedy Snake game for **Cocos Creator 3.x**.
The existing screenshot images from the original C++ project are reused here as
resources.  They are available under `assets/Resources` and can be added to
scenes for textures or placeholders.

## Opening the project

1. Install Cocos Creator 3.x.
2. Open this `cocos` folder as a project.
3. Attach the `GameManager`, `Snake`, and `Food` scripts to nodes as needed.

The TypeScript sources are located in `assets/Scripts` and should compile with
`tsc -p cocos` from the repository root.
